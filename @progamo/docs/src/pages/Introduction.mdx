# Design System para React com Stitches e Radix UI

Bem-vindo ao Design System para React com Stitches e Radix UI! Este projeto tem como objetivo fornecer um conjunto consistente de componentes e estilos para ajudar no desenvolvimento de interfaces de usuário eficientes e visualmente agradáveis.

## Sobre o Design System

Um Design System é uma coleção de padrões de design, componentes reutilizáveis, diretrizes de estilo e princípios de design que juntos fornecem uma base sólida para a criação de produtos e interfaces de usuário de alta qualidade. Com um Design System, é possível manter a consistência visual e de interação em todo o projeto, economizando tempo e esforço no desenvolvimento.

## Tecnologias Utilizadas

Neste projeto, utilizamos duas tecnologias principais:

- **Stitches:** Stitches é uma biblioteca CSS-in-JS que permite escrever estilos de forma mais modular e eficiente, com suporte para temas, componentes reutilizáveis e uma sintaxe simples e intuitiva.

- **Radix UI:** Radix UI é um conjunto de componentes acessíveis e de alto desempenho para React, construído com Stitches. Esses componentes são projetados para serem customizáveis e fáceis de usar, seguindo as melhores práticas de design e acessibilidade.
